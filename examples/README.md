# Generated output samples

Static examples of what `smartgen` commands produce (package name: `my_app`).

| Command | Example output |
|---------|----------------|
| `smartgen init` | [`smartgen.yaml`](smartgen.yaml) |
| `smartgen page profile --route` | [`lib/screens/profile_screen/`](lib/screens/profile_screen/) + [`lib/router/`](lib/router/) |
| `smartgen assets images` | [`lib/app/app_images.dart`](lib/app/app_images.dart) |

These files are for reference only — they are not a runnable Flutter app.
